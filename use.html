# 引入单文件组件

指定 src 即可，最好使用绝对路径，以免无法正确引入组件

# 批量引入

指定路径即可，fisandoc 会自动引入该路径下所有 .vue 文件

<demo-wrapper src="docs" />

结合 Vue2 使用

fisandoc 基于 vitepress。默认的 Vue 版本为 3.x。当你的项目是 Vue2.x时，且使用 composition-api demo 中 vue 需要改为 vue-demi。例如：

// import { ref } from 'vue'
import { ref } from 'vue-demi'
